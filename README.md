# DOD Tracker Mobile

Questo progetto e pronto per GitHub Pages.

## Pubblicazione rapida (da PC)

1. Crea un nuovo repository su GitHub.
2. Carica questi file (`index.html`, `.nojekyll` e gli altri file del progetto).
3. Vai su `Settings > Pages`.
4. In `Build and deployment` seleziona:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` (root)
5. Salva e attendi 1-3 minuti.
6. Apri il link generato (tipo `https://TUO-USER.github.io/NOME-REPO/`) da iPhone.

## Nota iPhone

Per evitare il viewer interno di app come Facebook/WhatsApp, apri sempre il link in Safari (o Chrome dopo apertura esplicita).
